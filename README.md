# fasterize demo

This repo contain 2 projects :
 - API: fasterize-api
 - Front: fasterize-front
 
## Install
To download all submodules please run this following command :

```
git clone --recurse-submodule https://github.com/emericspiroux/fasterize-demo.git
```
